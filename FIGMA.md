# 🎨 Макет в Figma

## Как создать макет приложения в Figma

### Ссылка на Figma:
**Создайте новый файл на [figma.com](https://figma.com) или присоедин итесь к существующему проекту**

### Компоненты дизайна:

#### 1. **Цветовая схема** (переменные в Figma)
```
Фон: #0b0f17
Панель: #121a2a
Текст основной: #eaf0ff
Текст приглушённый: #9fb0d0
Бренд (синий): #6ea8fe
Опасность (красный): #ff6b6b
```

#### 2. **Компоненты**

**Header (Заголовок)**
- Высота: 60px
- Лого: "PC Store" + подзаголовок "Комплектующие для ПК"
- Меню: Главная, Каталог, О магазине, Контакты, Корзина (с считателем)
- На мобилке (≤768px): гамбургер-меню вместо меню

**Главная страница**
- Hero: большой заголовок + описание + кнопка "Перейти в каталог"
- Категории: 4 карточки с иконами (2x2 на десктопе, 1x1 на мобилке)
- Преимущества: 4 карточки с галочками и текстом

**Каталог**
- Фильтры: поиск, выбор категории, сортировка
- Товары: сетка 3 колонки на десктопе, 1 колонка на мобилке
- Карточка товара: изображение, название, цена, рейтинг, 2 кнопки (Подробнее, В корзину)
- Пагинация: кнопки Назад/Вперед

**Модальное окно товара**
- Изображение слева, инфо справа (на мобилке - вертикально)
- Название, цена, описание
- Кнопки: В корзину, Закрыть

**Корзина**
- Список товаров с кол-вом и удалением
- Итоговая сумма справа (на мобилке - внизу)
- Кнопки: Оформить заказ, Очистить корзину

#### 3. **Типография**
- Заголовки (h1, h2): 900 font-weight
- Текст меню: 500 font-weight
- Основной текст: 400 font-weight

#### 4. **Отступы и сетка**
- Контейнер max-width: 1100px
- Горизонтальные отступы: 16px по краям на мобилке, 32px на десктопе
- Зазоры между элементами: 12-24px

#### 5. **Брейк-поинты для адаптива**
- Desktop: 980px+
- Tablet: 680px - 980px
- Mobile: до 680px

### Как начать работу в Figma:

1. **Создайте файл** → Выберите "Design file"
2. **Создайте Frame** для каждой страницы (1920x1080 для десктопа, 390x844 для мобилки)
3. **Добавьте цвета** как переменные (Assets → Colors)
4. **Создайте компоненты** для переиспользования (Header, Product Card, Button и т.д.)
5. **Сделайте версию для мобилки** скопировав фреймы и переделав layout

### Экспорт для разработчиков:

- Используйте плагин **Figma API** или **Design Tokens**
- Експортируйте компоненты как SVG/PNG для документации
- Делитесь ссылкой с правом просмотра для кодировщиков

---

**Совет:** Используйте Auto Layout в Figma для создания адаптивных компонентов!
